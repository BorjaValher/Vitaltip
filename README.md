# VitalTip 🩺 — Emergencia Médica NFC

> **Tu identidad de salud digital.** Una plataforma ligera y 'mobile-first' diseñada para salvar vidas mediante la sincronización instantánea de datos médicos críticos a través de tecnología NFC.

---

## 📝 ¿Qué es VitalTip?

**VitalTip** es una aplicación web diseñada para centralizar e identificar información médica crucial en situaciones de emergencia. Permite a los usuarios gestionar sus perfiles de salud (alergias críticas, grupo sanguíneo, medicación activa e historial clínico) y **vincularlos directamente a dispositivos físicos como pulseras, tarjetas o llaveros mediante tecnología NFC**.

En caso de accidente o emergencia médica, cualquier dispositivo compatible puede escanear el tag NFC para desplegar de forma inmediata el perfil clínico del paciente, optimizando los tiempos de respuesta del personal sanitario y reduciendo riesgos fatales.

---

## 🚀 Características Principales

* **Sincronización NFC Nativa:** Guarda y actualiza tus datos médicos esenciales directamente en tu dispositivo o pulsera inteligente con un solo toque.
* **Alertas Médicas de Alta Prioridad:** Visualización inmediata de alergias críticas y grupo sanguíneo en el modal de emergencia sincronizado.
* **Acciones SOS de un Clic:** Botones rápidos integrados para realizar llamadas de emergencia inmediatas y compartir la ubicación geográfica exacta en tiempo real.
* **Control de Medicación Activa:** Planificador diario inteligente organizado por turnos (mañana, tarde y noche) para el seguimiento de dosis.
* **Historial Clínico Digital:** Registro pormenorizado de eventos médicos, consultas generales, urgencias y cirugías previas.
* **Interfaz Limpia y UI/UX Intuitiva:** Diseño minimalista y moderno optimizado para pantallas móviles utilizando paletas de color de alta visibilidad (médica/alerta).

---

## 🛠️ Tecnologías Utilizadas

* **Frontend:** HTML5 estructural, Vanilla JavaScript (basado en arquitectura modular ES Modules) y **Tailwind CSS** para un estilado ágil y moderno mediante clases de utilidad.
* **Diseño e Iconografía:** Lucide Icons (`data-lucide`) para un set de iconos limpio y consistente en toda la interfaz.
* **Base de Datos y Autenticación:** Integración con **Supabase** (PostgreSQL en la nube y servicio de Auth), consumido de manera directa y segura en el cliente mediante CDN.
